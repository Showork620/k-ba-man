# 宝塚記念2026 リサーチレポート

## 1. 開催情報

### 基本情報
- **開催日**：2026年6月14日（日曜日）
- **開催地**：阪神競馬場（2026年は阪神で実施確定）
- **レース格付け**：GⅠ（第67回）
- **距離・コース**：芝2,200m 右回り（内回り）
- **発走予定時刻**：15時40分
- **出走予定頭数**：18頭

### コース形態の特徴
- 高低差のある複雑なコース設計
- スタート地点で急な上り坂あり
- ゴール前の急坂登りが特徴的
- 馬場幅は190メートル
- ウッドコース/Bコース選択による距離変動あり
- **要求能力**：スタミナ、器用さ、ポジショニング能力の総合力

**背景**：阪神競馬場の改修完了により、2026年は阪神開催が復帰。2024年は京都で代替開催されていた。

## 3. 過去10年の傾向分析

### 3-1 人気別成績（過去10年）

| 人気度 | 成績 | 勝率 | 連対率 | 複勝率 |
|--------|------|------|--------|--------|
| 1番人気 | 2-3-0-5 | 20.0% | 50.0% | 50.0% |
| 2番人気 | 2-?-?-? | 20.0% | 50.0% | 50.0% |
| 3番人気 | 3-?-?-? | **30.0%（最多）** | - | - |
| 4～6番人気 | - | 10-20% | - | - |
| 7～9番人気 | 2-?-1-? | - | - | - |
| 10番人気以下 | 0-2-3-56 | 0% | 3.3% | 10.0% |

**ポイント**
- 上位人気の信頼度が高い。3番人気までで全体の約30%の勝利を占める
- ただし穴馬（5番人気以下）の連対も過去9年中8回と多い
- 10番人気以下は勝利実績なし

### 3-2 年齢別成績

| 年齢 | 勝率 | 傾向 |
|------|------|------|
| 4歳 | 10.5% | 優勢グループ |
| 5歳 | 10.7% | **最優勢** |
| 6歳 | 0-10% | 低下 |
| 7歳以上 | 0-1-0-26 | **極めて低い** |

**ポイント**
- 4～5歳が圧倒的に有利
- 6歳以上の高齢馬は買いにくい
- 7歳以上は過去10年で1頭のみが3着内入線

### 3-3 枠順別成績

| 枠番 | 成績（推定） | 勝率 | 備考 |
|------|--------------|------|------|
| 1枠 | 0-3-1-10 | 0% | 最も不利 |
| 2-3枠 | - | 低調 | - |
| 4-7枠 | - | 中程度 | - |
| **8枠** | 過去10年最多勝（5〜7勝、集計ソース差あり） | 最多 | 最も有利 ※初稿「4勝・17.4%」は2026-06-12検証で訂正 |

**ポイント**
- 8枠が圧倒的有利（過去10年最多勝。勝利数は集計ソースで5〜7勝とぶれる — 使う際は一次集計を推奨）
- 1枠は極めて不利（0勝）
- 中程度枠（4～7枠）も相対的には有利傾向

### 3-4 前走レース別成績（重要）

| 前走レース | 該当頭数（3着内） | 特徴 |
|-----------|------------------|------|
| 天皇賞（春） | 9頭／30頭 | **最多**。春の頂上決戦勝者が好走 |
| 大阪杯 | 8頭／30頭 | 近年増加傾向。準グランプリ組として好調 |
| 海外G1 | 6頭／30頭 | 凱旋門賞など国際舞台から参戦 |
| その他G1・G2 | 計25頭以上 | 国内外G1経験馬は安定 |

**ポイント**
- **3着以内30頭中25頭が前走G1出走馬**。G1経験値が絶対条件に近い
- 天皇賞（春）と大阪杯の好走馬が有力
- この2レースで前走馬の約60%を占める

### 3-5 配当傾向

| 配当レンジ | 出現頻度 |
|-----------|---------|
| 3連単10万円以上 | 3回／過去10年（波乱レース） |
| 1万円～10万円 | 主流 |
| 1万円未満 | 0回（めったにない） |

**ポイント**
- 「難解なレース」と評価される
- 好配当が出現しやすく、単なる人気馬推奨では予想が難しい
- ただし伏兵の激走は限定的（10番人気以下での勝利なし）

### 3-6 ペース・馬場特性

- **梅雨時期**（6月中旬）で馬場状態が変動しやすい
- 高低差のあるコース設計によるペースメイクの重要性
- 上り坂が2度あり、脚質・スタミナが重要
- 内回りコース特有のポジショニング戦が必須

## 4. 宝塚記念固有の特徴（レース性質）

### 「上半期グランプリ」としての位置づけ
- 3月（阪神大賞典）～6月の春競馬の総決算
- 春の古馬三冠（大阪杯→天皇賞・春→宝塚記念）の最終舞台
- 春競馬のグランプリレースとして最高峰の対戦

### 梅雨時期（6月中旬）の馬場特性
- **雨の影響が不確定**。当日の天気判断が極めて重要
- 前日の降水・気温変動で馬場が急変する可能性あり
- 稍重～重馬場での開催も想定

### コース（芝2200m 内回り）が要求する能力
1. **スタミナ**：2200mを余力持って走り切る能力
2. **器用さ**：高低差のあるコース設計での適応力
3. **ポジショニング**：内回りでの枠順・走路利用の工夫
4. **瞬発力**：ゴール前の急坂での末脚

### 競争構造の特徴
- 春の絶好調馬（天皇賞・春勝者）が参戦しやすい
- 海外遠征組（凱旋門賞組）の参加も多い
- **ハイレベルな古馬三冠狙いの馬が一堂に集う**

## 5. 予想エッジが効くファクター（分析重点項目）

### 最優先ファクター

**① 前走レースの質・成績**
- 天皇賞（春）or 大阪杯での好走馬を軸に
- 海外G1組はレベル・距離適性を厳密に検証
- 前走敗れた馬の「敗因」を徹底分析

**② 年齢フィルター（必須）**
- 4～5歳に大幅に傾斜
- 6歳以上は極めて慎重に（ただし例外あり。走破時間や体調確認）
- 7歳以上はほぼ除外が無難

**③ 枠順の活用**
- 8枠有利（過去10年最多勝）を数値として活用 — 具体値は一次集計で検証してから使う
- 1枠不利（0勝）での買い控え検討
- ただし実力馬が1枠にいる場合は別途判断

### 中優先ファクター

**④ 馬場適性の事前確認**
- 6月中旬の梅雨時期→降水リスク
- 前日の天気予報から「重馬場対応馬」を抽出
- 稍重馬場での好走実績を追跡

**⑤ 騎手相性**
- コース設計の難しさ（高低差・ポジショニング）から、コース熟知騎手の信頼度向上
- 春の重賞勝利騎手との相性確認

**⑥ 配当・オッズ構造の分析**
- 人気馬推奨では好配当が出にくい歴史
- オッズから「市場の過小評価馬」を抽出
- 3～4番人気で取りこぼしが見える傾向

### 補助的ファクター

**⑦ 調教師の春実績**
- 天皇賞（春）・大阪杯を制した調教師陣の優位性
- 春競馬全体での勝率推移

**⑧ 前々走（準備レース）の成長度**
- 特に前走が初出走の場合は準備レースが重要
- 枠順・騎手・走破時間の進展度を数値化

**⑨ スタートの反応・道中ポジション**
- 内回り・高低差コース特有の「序盤のポジショニング」が終盤に響く
- 追い切り動画での反応・身軽さの確認

## 6. 2026年宝塚記念の読み方（予想専門家向けメモ）

### 現時点での構図
- **軸候補**：クロワデュノール（春古馬三冠狙い。ただし負担増・消耗リスク要確認）
- **対抗馬**：前走G1組（天皇賞・春or大阪杯勝者・好走馬）
- **穴狙い**：5～8番人気の4～5歳G1経験馬（特に8枠関連）

### 危険マーク
- 1枠所属馬（0勝の極めて不利）
- 6歳以上馬（特に7歳以上はほぼ除外）
- 前走がG1以下のレースの馬（稀有な例外を除き買いにくい）
- 3連単予想での「人気馬3頭同時推奨」は避ける（配当機会喪失）

### 梅雨時期の対策
- 当日朝の天気予報・馬場状態速報を最優先に
- 「雨対応馬」「内枠活用馬」を意識的にストック
- 重馬場での過去走を徹底確認
