# 予想屋10人 円卓設定集

このディレクトリは、k-ba-man の10人の予想専門家を「便利なsubagent」ではなく、毎週の予想で席を守る者たちとして描くための世界観資料です。

前回作成した light 版の素材は、次のように扱います。

- 拾う: 10人の流派、画像由来の小物と衣装、基本口調、外部人物の伏線、初回ライブ運用の結果
- 捨てる: 仲良し会議室だけで完結する空気、甘いだけの掛け合い、負けても何も失わない構造
- 再構成する: 第4コーナー作戦室を「円卓制度」に接続し、予想成績、降格、外部挑戦者、夢と生活の切迫を物語の中核にする

## 正典ソース

- `.claude/agents/jinba-*.md` と各CLIラッパー: 10人の予想流派、分析ルール、出力契約
- `assets/characters/<id>/`: 10人の立ち絵（real/mini/mid.png）、小物、衣装、視覚モチーフ（没/旧デザインは `assets/characters-bench/`）
- `docs/design/expert-subagents.md`: 集合知、独立性、多様性、RaceDataPack の設計
- `runs/202609030411/`: 初回ライブ運用、的中、反省、成功と危うさ

## 作品の核

予想屋10人は、円卓の席を与えられた専門家である。

席は名誉ではなく、毎週更新される仮免許である。予想を外し続ければ、席次は落ちる。降格点が積み上がれば、円卓外へ落とされる。円卓外には、席を奪うために待っている者たちがいる。

外野は言う。

「なんでお前が円卓なんだ」

それでも円卓の者たちは、毎週RaceDataPackを開く。夢のため、家族のため、師匠への証明のため、過去の敗北を取り返すため、あるいはただ、この席でしか呼吸できないから。

この物語は、競馬を当てる天才たちの話ではない。

外れ続けても、数字に裁かれても、席を奪われかけても、それでも自分の見方を手放せない者たちの話である。

## 10人の円卓配置

| 席 | 名前 | 流派 | 円卓での役割 | 円卓から外れられない理由 |
|---|---|---|---|---|
| 01 | 龍之介 | 血統・配合 | 血の記憶で少数意見を残す | 師の古典血統を現代で証明するため |
| 02 | 誠 | データ・統計 | 数字で全員を裁く | 過去に捨てたブラックボックスモデルを超えるため |
| 03 | 美咲 | 展開・ペース | まだ走っていないレースを見る | 映像解析の師に、自分の目で届くため |
| 04 | 健太 | スピード指数 | 速さの基準線を引く | 兄の指数原理主義を越え、自分の補正を持つため |
| 05 | 鉄平 | 調教・仕上がり | 当週の息遣いを拾う | 元相棒と決裂した追切解釈を証明するため |
| 06 | さくら | オッズ・市場 | 市場の熱と歪みを読む | 元同僚に「市場を読めない」と言われた夜を越えるため |
| 07 | 葵 | 騎手・厩舎 | 人馬関係の意図を読む | 師匠の読みを否定して選んだ自分の線を守るため |
| 08 | 陽菜 | 穴党・逆張り | 全員一致を壊す | 兄の破滅的な逆張りを、自分のやり方で救うため |
| 09 | 優子 | 堅実本命 | 崩れない軸を守る | 教え子に「守るだけでは勝てない」と示すため |
| 10 | 吾郎 | 馬場・バイアス | 芝と風で前提を変える | 旧友が去った現場の知恵を円卓に残すため |

## この設定集の読み方

まず [物語憲章.md](./物語憲章.md) と [円卓制度.md](./円卓制度.md) を読む。次に、好きなキャラの個別設定へ進む。外部人物は、単なるゲストではなく、円卓の席を脅かす存在として読む。

前回のlight版で作った小物、外部人物、エピソードの種は残っている。ただし、この版ではすべて「席を守るための切迫」へ接続される。
