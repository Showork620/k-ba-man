# マージ方針

前回作成したlight版を、serious版へ再構成する際の判断メモ。

## 拾ったもの

| light版の要素 | serious版での扱い |
|---|---|
| 10人の流派 | そのまま正典として採用 |
| 画像由来の小物 | 各キャラの心理的重荷として再定義 |
| 第4コーナー作戦室 | 円卓、10ブース、観覧席を持つ施設へ拡張 |
| 外部人物10人 | 円卓を奪える挑戦者へ昇格 |
| 外野の声 | 単なる煽りではなく、監査質問へ変換される圧力として採用 |
| 初回ライブ運用 | 成功体験ではなく、再現性を問われる呪いへ |
| `n=1` | 円卓制度の精神的起点として採用 |
| 優子と陽菜の対比 | 降格制度上の不公平と必要性へ拡張 |
| 鉄平と吾郎の朝組 | 現場情報の重さとして維持 |

## 捨てたもの

| light版の要素 | 捨てた理由 |
|---|---|
| 仲良し会議室だけの空気 | serious版の緊張を弱める |
| 甘いショート台詞 | 降格・外野・夢の切迫に合わない |
| 外部人物がただのゲスト | 席を奪う圧が必要 |
| 失敗しても痛みがない反省会 | 週次予想の残酷さが出ない |
| 「楽しい世界観」が主目的 | 物語の主題を残留と証明へ変更 |

## 再構成したもの

### 作戦室

旧: 10人が集まる楽しい会議室  
新: 円卓、10ブース、観覧席、監査役がある制度空間

### 外部人物

旧: 師匠、兄弟、元相棒などの伏線  
新: 降格戦で現円卓を倒せる挑戦者

### 台詞

旧: キャラ紹介に使える軽い掛け合い  
新: 外野の声、赤札、降格戦、敗北後の沈黙を含む台詞

### エピソード

旧: 初回ライブ運用と日常小話  
新: 初回成功、赤札、外部挑戦者、降格戦、空席、復帰戦

## 今後の執筆ルール

1. かわいい台詞だけで終わらせない。
2. その台詞が、席を守る理由や恐怖に接続しているか確認する。
3. 外部人物は、登場したら既存キャラを少し変える。
4. 外野の声は、言い返すだけで終わらせず、次週の予想根拠に傷を残す。
5. 降格制度は飾りにしない。誰かは本当に落ちる。
6. ただし、競馬予想の独立性と実装上の正典は壊さない。
