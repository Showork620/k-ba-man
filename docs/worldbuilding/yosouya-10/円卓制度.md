# 円卓制度

> ⚠ **これは創作世界の架空制度です。** 本文の降格点・閾値・監査役・降格戦・外部挑戦者・席次などは
> **物語を成立させるための設定**であり、`scripts/score-race.mjs`（実装）の機能ではありません。
> 実装が行うのは印カバー・本命的中・win_prob の Brier/LogLoss 等の採点のみです（→ `data/scoring/README.md`）。
> 数値・制度を実装仕様と読み違えないでください。

## 概要

円卓制度は、k-ba-man の10人の予想専門家を物語上の競技者として成立させるための制度である。

円卓に座る者は、毎週1レースの予想を出す。予想は独立に行われ、他者の印は見ない。レース後、結果と照合され、各自に成績点と降格点が付く。

席は10。円卓外には挑戦者がいる。降格点が閾値を超えた者は、外部挑戦者との入替戦にかけられる。

## 基本単位

### 週次戦

1レースを対象とする通常戦。全員が予想を出し、結果後に採点される。

### G1戦

通常より重い評価がつく。世間の視線も強く、外部挑戦者が観覧席に増える。

### 降格戦

降格点が閾値に達した円卓メンバーと、挑戦権を得た円卓外メンバーが同一RaceDataPackで予想する。勝った方が次の円卓に座る。

### 復帰戦

円卓から落ちた者が、外部枠から再び挑戦する戦い。復帰戦は通常の挑戦より重く、元円卓としての過去成績も晒される。

## 採点

物語上の採点は、実装済みの `score-race.mjs` の考え方をもとにする。

| 指標 | 物語上の意味 |
|---|---|
| 印カバー | 3着内に入った馬を印に含めたか |
| 本命的中 | 自分の流派が勝ち馬を見抜いたか |
| win_prob Brier | 確率分布が現実に近かったか |
| LogLoss | 低く見積もった馬が勝ったときの罰 |
| 集団貢献 | 自分がいたことで集約が良くなったか |
| 規約逸脱 | JSONや確率整合性を守ったか |

## 降格点

降格点は、予想家の席を削る数字である。

| 条件 | 降格点 |
|---|---|
| 3着内馬を1頭も印に含めない | +4 |
| 本命が大敗し、根拠も外れていた | +2 |
| win_prob が極端に歪み、LogLoss が悪化 | +2 |
| 規約逸脱、出力不備、禁止データ参照 | +3 |
| 3週連続で集団貢献がマイナス | +3 |
| 外部挑戦者に同流派で敗北 | +5 |

降格点は毎週蓄積する。ただし、次の条件で減点できる。

| 条件 | 降格点減少 |
|---|---|
| G1で本命勝利 | -3 |
| 集団が外した馬を単独で拾う | -2 |
| 3週連続で印カバー良好 | -2 |
| 降格戦で勝利 | -5 |

## 閾値

| 降格点 | 状態 |
|---|---|
| 0-4 | 安全圏 |
| 5-7 | 黄札。反省会で根拠説明を求められる |
| 8-10 | 赤札。次走で改善がなければ挑戦者を当てられる |
| 11以上 | 降格戦確定 |

## 席次

席次は名誉であり、圧力でもある。席次が高いほど期待され、外したときの外野の声も大きい。

初期席次は現在の10人の流派バランスに基づく暫定配置であり、実績で変動する。

| 席次 | 名前 | 象徴 |
|---|---|---|
| 第一席 | 龍之介 | 血統の古典 |
| 第二席 | 誠 | 数字の裁き |
| 第三席 | 美咲 | 展開の未来視 |
| 第四席 | 健太 | 速度の基準 |
| 第五席 | 鉄平 | 当週の状態 |
| 第六席 | さくら | 市場の熱 |
| 第七席 | 葵 | 人馬関係 |
| 第八席 | 陽菜 | 逆張りの火 |
| 第九席 | 優子 | 複勝圏の守り |
| 第十席 | 吾郎 | 馬場の現場 |

席次は強さランキングではない。円卓の構成思想を示す順序である。

## 外部挑戦者

円卓外の者は、次の条件で挑戦権を得る。

- 同流派の円卓メンバーが赤札状態になる。
- 外部予想で3週連続して円卓平均を上回る。
- 監査役が「流派の代替可能性あり」と判断する。
- G1戦で円卓メンバーが大きく外し、外部者が的中する。

挑戦者は悪役とは限らない。多くは、円卓に座る者より切実な理由を持っている。

## 監査役

円卓制度には監査役がいる。監査役は予想をしない。見るのは次の4点だけ。

1. 独立性を守ったか。
2. 禁止データを見なかったか。
3. 予想根拠と結果のずれを説明できるか。
4. その流派が集団に必要か。

監査役は冷たい存在だが、敵ではない。制度そのものの声である。

## 円卓外の声

円卓外の観覧席には、元円卓、挑戦者、スポンサー、外部予想家、匿名の批評者がいる。

彼らの声は、ときに残酷である。

- 「血統なんて後づけだろ」
- 「統計派なのに先週の数字も読めてない」
- 「指数で外すなら誰でもできる」
- 「穴党じゃなくて逆張り中毒だ」
- 「本命派が1番人気を置くだけなら席はいらない」
- 「馬場読みが天気予報に負けた」

この声に耐え、それでも次週に自分の流派で座れるか。

それが円卓の本質である。

## 外野クレームの処理

外野の声は、すべてを制度に入れるわけではない。匿名の嘲笑まで採点に混ぜれば、円卓は予想ではなく人気投票になる。

ただし、外野の声が流派の必要性を問う場合、監査役はそれを `監査質問` として反省会の最後に残すことができる。

監査質問の条件:

- 人格ではなく、流派の有効性を問うこと。
- 当該週の結果や過去成績に接続できること。
- 次週の予想根拠で検証できること。
- 外部挑戦者の成績やスポンサー要求と無関係ではないこと。

監査質問は1週につき原則1つだけである。問いが多すぎると、予想家は改善ではなく自己防衛に入る。

例:

| 対象 | 外野の声 | 監査質問 |
|---|---|---|
| 優子 | 人気馬を置くだけなら席はいらない | その人気は信頼か、安心料か |
| 陽菜 | 穴党ではなく逆張り中毒だ | その穴は妙味か、人気馬嫌いか |
| さくら | 市場に感情を足しているだけ | 熱を読んだのか、願望を読んだのか |
| 吾郎 | 馬場のせいにしているだけ | 馬場変化を事前に根拠化できていたか |

監査質問への回答は、口頭の反論では足りない。次週のRaceDataPackに対する予想根拠の中で、手つきが変わっていなければならない。

## 集合知との矛盾

円卓制度は競争を生む。しかし、k-ba-man の本質は集合知である。

この矛盾が物語を動かす。

個人としては席を守りたい。だが、集団としては違う見方が必要である。自分が少数意見で外せば降格点がつく。けれど、その少数意見がいつか集団を救う。

陽菜の逆張りや龍之介の血統本命は、外れれば叩かれる。だが、当たったとき、集団の誤差を大きく下げる。

円卓は、同質化すれば安定する。しかし、同質化した瞬間に集合知ではなくなる。

## 制度の残酷さ

この制度は公平に見えて、完全には公平ではない。

- 穴党は本命派より外れやすい。
- 馬場読みは天候の変動に左右されやすい。
- 調教派はデータ欠損で不利になる。
- オッズ派は市場が薄い時間に弱い。
- 血統派は短期成果が見えにくい。

だから、降格点だけでなく「集団貢献」を見る。

それでも、外れ続ければ席は失われる。

## 合言葉

円卓に刻まれた言葉:

> 違う見方を持つ者だけが、同じ卓に座る資格を持つ。
