# 既存IP研究

この資料は、既存IPの表層を借りるためではなく、物語構造を抽出して「予想屋10人 円卓設定」に翻訳するためのメモである。

## 参照方針

参考にするのは固有名詞、キャラクター、設定のコピーではない。

参考にするのは次の構造である。

- 競争制度が人格を追い詰める構造
- 席、階級、ランキング、降格
- 夢と生活がかかった勝負
- 外部からの挑戦者
- 仲間であり競争相手でもある関係
- 勝った後にも残る苦味

## 『ブルーロック』から抽出するもの

公式イントロダクションでは、300人の高校生FWが世界一のストライカーを目指し、己のサッカー生命とゴールをかけて挑む「デスゲームのような作風」と説明されている。また「世界一のエゴイストでなければ、世界一のストライカーにはなれない」という価値観が掲げられている。参照: [劇場版ブルーロック -EPISODE 凪- 公式](https://movie.bluelock-pr.com/)

抽出する構造:

- 選抜施設
- 生存競争
- 才能を順位で可視化する制度
- 脱落がキャリアの死に近い
- 個人のエゴがチーム戦に混ざる

円卓への翻訳:

- 円卓席次は、予想家としての生命線。
- 降格は「便利なAIの停止」ではなく、流派ごと否定される恐怖。
- 10人は集合知の仲間だが、席を守るためには自分の流派の価値を証明しなければならない。

## 『賭ケグルイ』から抽出するもの

公式イントロダクションは、ギャンブルには勝者と敗者が存在し、勝者は富と栄誉を得て、敗者は落伍者の烙印と喪失を味わうと語る。第2期公式では、ギャンブルで支配された名門校、生徒会、弱肉強食、究極の選挙バトルロイヤルが示される。参照: [賭ケグルイ1期公式](https://kakegurui-anime.com/1st/introduction/), [賭ケグルイxx公式](https://kakegurui-anime.com/introduction/)

抽出する構造:

- 勝敗が階級を作る
- 欲望が勝負を美しくも醜くもする
- 敗者には社会的烙印がある
- 選挙や投票が戦場になる

円卓への翻訳:

- 予想成績は席次と階級を作る。
- 円卓外の者は「外された者」として見られる。
- 外部挑戦者は、単に席が欲しいだけでなく、円卓の価値観を壊しに来る。

## 『カイジ』から抽出するもの

『カイジ』では、借金を背負った主人公がギャンブルへ誘われ、勝負のたびに負債、身体的損失、地下労働などの極限へ追い込まれる。概要では、船、ホテル、地下労働、限られた日数での大金獲得など、勝ってもなお抜けられない苦境が示されている。参照: [Kaiji 概要](https://en.wikipedia.org/wiki/Kaiji_%28manga%29)

抽出する構造:

- 借金や負債による強制参加
- 勝負が生活と身体を削る
- 勝ってもすぐには救われない
- 裏切りと信頼の揺れ

円卓への翻訳:

- 予想屋たちは全員、何かしらの「負債」を抱えている。金銭とは限らない。
- 降格点は心理的な借金である。
- 初回成功しても、n=1という不安が残る。

## 『LIAR GAME』から抽出するもの

『LIAR GAME』では、参加者が大金と負債を背負い、嘘や駆け引きが推奨されるゲームを進む。主人公たちは勝つだけでなく、負債を背負った他者を救い、組織を内側から崩そうとする。参照: [LIAR GAME 概要](https://en.wikipedia.org/wiki/Liar_Game)

抽出する構造:

- 参加者同士が敵でもあり、救済対象でもある
- ルールの穴を読む
- 透明な善意が弱点にも武器にもなる
- 主催者の意図が常にある

円卓への翻訳:

- 円卓制度にも主催者と監査役がいる。
- 10人は他者を蹴落とすだけではなく、流派ごと救おうとする。
- ルールをどう読むかが、予想と同じくらい重要になる。

## 『バクマン。』から抽出するもの

『バクマン。』は、漫画家を目指す2人が連載、打ち切り、ライバル、過労、夢への約束に向き合う物語である。概要では、初連載の打ち切り、次作の挫折、ライバルとの競争、アニメ化への夢が示されている。参照: [Bakuman 概要](https://en.wikipedia.org/wiki/Bakuman)

抽出する構造:

- 毎週の成果物
- 人気と順位による評価
- 打ち切りの恐怖
- 夢と仕事の境界
- ライバルが成長の基準になる

円卓への翻訳:

- 毎週の予想は「連載回」。
- 外し続けることは打ち切りに近い。
- 円卓外の挑戦者はライバルであり、次の席を狙う連載候補。

## 『あかね噺』から抽出するもの

VIZ公式は『あかね噺』を「Climbing the Ranks of Rakugo」として紹介し、巻ごとに観客、ライバル、師弟、流派の歴史が積み上がる構造を示している。参照: [VIZ Akane-banashi](https://www.viz.com/akane-banashi)

抽出する構造:

- 芸の階級
- 師弟関係
- 破門、昇進、名跡
- 同じ演目でも演者で意味が変わる

円卓への翻訳:

- 予想流派を「芸」として扱う。
- 外部人物は師匠、兄弟子、破門者、別流派の演者として機能する。
- 同じRaceDataPackでも、誰が読むかでまったく違う演目になる。

## 『ちはやふる』から抽出するもの

『ちはやふる』は、競技かるたというニッチな題材を、個人の夢、団体戦、師弟、ライバル、再会の物語に変換している。参考レビューでも、仲間を集めて高校チームとして日本一を目指す構造、個人の夢とチーム戦の両立が語られている。参照: [Chihayafuru 参考レビュー](https://www.reddit.com/r/anime/comments/331qz4/wt_chihayafuru/)

抽出する構造:

- ニッチ競技を熱く見せる
- 個人戦と団体戦の両立
- 夢を追う者、失った者、戻ってくる者
- 対戦相手にも人生がある

円卓への翻訳:

- 競馬予想という題材を、予測手法の競技として見せる。
- 10人は個人戦として予想し、集約で団体戦になる。
- 外部挑戦者にも、円卓を奪うだけの人生がある。

## 統合設計

円卓設定は、次の混合で作る。

| 参照構造 | 円卓での実装 |
|---|---|
| ブルーロックの選抜 | 席次、降格、外部挑戦者 |
| 賭ケグルイの階級 | 円卓/円卓外/観覧席/挑戦権 |
| カイジの負債 | 各キャラの「降りられない理由」 |
| LIAR GAMEの心理戦 | 監査役、制度の穴、外部勢の策略 |
| バクマンの毎週評価 | 週次予想、連敗、打ち切り感 |
| あかね噺の師弟と芸 | 流派、師匠、破門、名跡 |
| ちはやふるの個人/団体 | 個別予想と集約の二重構造 |

## この研究からの結論

予想屋10人は、仲良しの専門家集団では足りない。

彼らは毎週、次の問いに答えなければならない。

「お前の見方は、まだ円卓に必要か」

この問いがある限り、血統も、統計も、展開も、指数も、調教も、オッズも、騎手も、穴も、本命も、馬場も、ただのキャラ属性ではなくなる。

それぞれが、生き残るための武器になる。
