## v1 との差異：なぜ別案が必要か

v1「深夜の円卓、かすかな光」は VTuber 的な没入感を競馬予想の世界に持ち込む提案だった。  
キャラカラーによるグロー・カード浮き上がり・フルブリードスライダーは、ぶいすぽっ！やメメントモリに学んだパターンだ。

v2 が目指すのは、別の問いへの答えである。

> **「競馬の格式感を正面から取り上げたら、どうなるか？」**

競馬には VTuber が持っていないものがある——新聞の活字、馬券の印刷物、競馬場の静寂と興奮。  
10人の予想屋を「VTuberに似た何か」ではなく、「腕利きの競馬評論家たち」として見せる。  
サイトの質感は、高級競馬クラブの会員誌に近いものにする。

---

## コンセプト：「黒革の会員誌」

**テーマ**: 高級競馬クラブの会員誌 × デジタル

豪奢なクラブハウスの受付に置かれた、表紙が黒革の会員名簿。  
ページを開くと、金の活字で予想家の名前が並んでいる。  
データは「数字」ではなく「読み物」として配置されている。

visitorsはそこに「分析者の声を読む」体験をしに来る。

## ビジュアルトーン

### カラー

| 役割 | 値 | 意図 |
|------|----|------|
| ベース背景 | `#0c0c0d`〜`#111113` | v1 より黒に近い。印刷用紙の裏面的な漆黒 |
| カード・パネル背景 | `#161617`〜`#1c1c1e` | セクションの奥行き分離 |
| ゴールド（主役） | `#c8a96e` | 見出し・ルール・ボーダーのすべてに使う |
| ゴールド（暗） | `#8a7040` | サブテキスト・下層の装飾 |
| テキスト（主） | `#f0d8bc` | 温かみのあるオフホワイト。ウォームトーン |
| テキスト（副） | `#a89880` | キャプション・日付・補足 |
| キャラカラー | `var(--accent)` | 控えめに使う。細ライン・シートナンバーのみ |
| バーガンディ（任意） | `#5c2230` | 1位・本命・特別行の強調アクセント |

**v1 との違い**: v1 はキャラカラーが主役でゴールドが添え物。v2 はゴールドが主役でキャラカラーが識別子にとどまる。

### フォント

| 用途 | 推奨 | 理由 |
|------|------|------|
| 見出し（英語・数字） | EB Garamond / Libre Baskerville | 印刷物的な格式感。無料で品質が高い |
| 見出し（日本語） | Shippori Mincho B1 | 源ノ明朝より細身で新聞的。Google Fonts 収録 |
| ラベル・ナビ | EB Garamond（大文字、letter-spacing 0.25em） | `ANALYSTS` `RACES` `RANKING` を活字組版風に |
| 本文（日本語） | Noto Serif JP（Light〜Regular） | セリフ体で統一し、雑誌本文感を出す |
| 数値統計 | EB Garamond（tabular-nums） | 回収率・的中数を「会計帳簿」風の数字に |

> v1 は本文に Noto Sans JP（サンセリフ）を使ったが、v2 は本文までセリフ体で通す。  
> サンセリフを排除することで「新聞/雑誌の紙面」的な統一感が生まれる。

### テクスチャ・装飾

- **紙目テクスチャ**: `#0c0c0d` ベースに `opacity: 0.03〜0.04` の noise を重ねる（v1 より密度を落とす）
- **細いゴールドルール**: `border: 1px solid rgba(200, 169, 110, 0.35)` をセクション区切りに多用
- **上下ラインの見出し**: 見出しの上下に細いゴールドラインを添える（競馬新聞的な区切り）
- **グロー効果は使わない**: v1 のグロー（`box-shadow: 0 0 20px var(--accent)`）は廃し、格式感を優先

### トップページ (Overview)

#### コンセプト：雑誌の表紙

```
[ヘッダー — 固定]
  左: ブランドロゴ（K）ゴールド
  右: ANALYSTS / RACES / RANKING（大文字・間隔広め）

[マストヘッド — 雑誌表紙風フルスクリーン]
  背景: 漆黒（#0c0c0d）+ 極薄ノイズ
  中央上部: ゴールドで「k-ba-man」ロゴタイプ（Shippori Mincho）
  中央: 大きなゴールドルール（1px 水平線、全幅）
  ルール下: 今週のフィーチャーコピー（例: 「宝塚記念、10人の答え合わせ。」）
  右下: 直近レース結果サマリー（回収率 + 的中者数）を小さく
  背景装飾: 馬のシルエット or lineup 画像を 5% 透明度で極薄く敷く

[予想家セクション — エディトリアルグリッド]
  見出し: 「THE ANALYSTS」（大文字、ゴールドライン付き）
  グリッド: 3〜4カラム（モバイル: 2カラム）
  各カード: プロフィール写真風の縦長矩形 + 名前（日英）+ 流派タグ
  ← v1 の「立ち絵を大きく見せる」と異なり、顔や上半身の肖像写真スタイル

[最新レースセクション — 記事ヘッド風]
  見出し: 直近レース名（例: 「2026 宝塚記念」）を雑誌記事の見出し組版で
  本文頭: 「回収率 171.7%。龍之介が血統でトップを取った週。」
  → 数値を「記事の書き出し」として見せる

[フッター]
  細いゴールドルールで区切り
  10人のシートナンバーを等間隔に横並び（アイコンではなく番号テキスト）
  「円卓は今週も開かれている」的なコピー
```

### キャラクター一覧ページ (Analysts)

#### コンセプト：雑誌の寄稿者一覧ページ

```
[ページヘッド]
  タイトル: 「THE ANALYSTS」大文字・ゴールドルール付き
  サブテキスト: 「10人の予想家が、それぞれの流派で円卓に座る」

[グリッド — 5カラム（PC）/ 2カラム（モバイル）]
  [各カード]
    上部: キャラ画像（縦長、object-fit: cover）
           ← v1 と同様だが、カードの雰囲気が違う
    区切り: ゴールドの細い水平ライン
    シートナンバー: 「No. 01」形式（ゴールド、小さく左寄り）
    名前: 大きく、Shippori Mincho
    流派: 大文字ラベル（例: 「BLOODLINE ANALYST」）
    
    ホバー時:
      - カードが浮く代わりに、ゴールドボーダー（1px → 2px）が強くなる
      - 名前下に格言/セリフが1行フェードイン（フォントはイタリック）
      - v1 のグローエフェクトは使わない
    
    カード背景:
      通常: #161617
      ホバー: #1c1c1e（少し明るくなるのみ。キャラカラーは使わない）
```

**v1 との差異**:  
v1 は「ホバーでキャラカラーのグローが縁に出る」が、v2 は「ゴールドボーダーが太くなる」だけ。  
全員がキャラカラーで光る v1 は華やかだが、v2 はモノクロームのアクセントで格式感を保つ。

### 個別キャラクター詳細ページ (Analyst Profile)

#### コンセプト：雑誌の人物インタビュー見開き

```
[プロフィールヒーロー — 雑誌見開き風]
  左半分（テキストカラム）:
    - 「No. 01」シートナンバー（小さく、ゴールドルールの上に）
    - 名前（大きく、Shippori Mincho、縦に大きく）
    - 流派ラベル（英語大文字）
    - セリフ（pull quote 形式）:
        フォントサイズ 2〜2.5rem、イタリック、
        左に縦のゴールドライン（v1 と同じ発想だが、太さは細め 2px）
        「セリフ」が読み物として機能するように組版
    
  右半分（ビジュアルカラム）:
    - キャラ画像（*-real.png を優先）
    - 画像の下に向かって漆黒グラデーション
    - 画像の背景: キャラカラーを 8% 透明度のみ（v1 は 30%）
  
  背景全体: 漆黒（#0c0c0d）

[スクロール後: プロフィールセクション — 記事本文風]
  2カラム（PC）:
    左カラム（本文）:
      intro + drama テキストを記事として読ませる
      「見出し1：」「見出し2：」形式でブレイクアップ
      統計を本文中に inline で言及する（「直近 20 戦で回収率 171%」など）
    右カラム（サイドバー）:
      「DATA BOX」スタイルのサイドバー
      回収率・的中率・平均順位の4タイルを細いゴールドボーダーで囲む
      流派タグ（強み）を箇条書きで

[成績履歴テーブル — 紙面の記録欄風]
  テーブルのヘッダー行にゴールドルール（上下 1px）
  行の奇偶で極薄の背景差（#111113 / #141416）
  ホバー行: ゴールド 5% 透明度の背景（v1 はキャラカラー）

[付加コンテンツ（後フェーズ）]
  「直近のコメント」セクション（各レース後の自動生成コメント）
  → 雑誌の「評論家コラム」的な位置づけ
```

#### コンセプト：競馬新聞の「印付き馬」欄

## CSS 設計ヒント（v2 固有）

v1 の CSS ヒントに追加・差し替えるもの。

```css
/* 雑誌風の大文字ラベル */
.label-caps {
  font-family: 'EB Garamond', serif;
  text-transform: uppercase;
  letter-spacing: 0.25em;
  font-size: 0.75rem;
  color: #c8a96e;
}

/* ゴールドルール（水平） */
.rule-gold {
  border: none;
  border-top: 1px solid rgba(200, 169, 110, 0.35);
  margin: 1.5rem 0;
}

/* 見出し上下ライン装飾 */
.heading-ruled {
  padding: 0.5rem 0;
  border-top: 1px solid rgba(200, 169, 110, 0.5);
  border-bottom: 1px solid rgba(200, 169, 110, 0.5);
}

/* Pull Quote（セリフ） */
.pull-quote {
  font-family: 'EB Garamond', serif;
  font-style: italic;
  font-size: 1.75rem;
  line-height: 1.5;
  color: #f0d8bc;
  border-left: 2px solid #c8a96e;
  padding-left: 1.5rem;
  margin: 2rem 0;
}

/* 統計数値：会計帳簿風 */
.stat-value {
  font-family: 'EB Garamond', serif;
  font-feature-settings: "tnum";  /* tabular-nums でカラム揃え */
  font-size: 2rem;
  color: #c8a96e;
}

/* 日本語：プロポーショナルかな */
.ja-text {
  font-family: 'Noto Serif JP', serif;
  font-feature-settings: "palt";  /* 日本語プロポーショナルかな */
}

/* キャラカラーは識別子のみ（細ライン） */
.char-accent-border {
  border-left: 3px solid var(--accent);  /* v1 の 4px より細め */
}
```

## 実装フェーズ提案（v2）

| フェーズ | 内容 | v1 との差分 |
|----------|------|------------|
| 1 | フォント差し替え（Shippori Mincho B1 + EB Garamond 導入）+ カラートークン整備 | v1 はこのフェーズで Playfair 導入 |
| 2 | ゴールドルール・ラベルキャップス・テーブルスタイルの統一 | v1 はカードホバーグロー |
| 3 | 詳細ページを「雑誌見開き」レイアウトへ。pull-quote セリフを前面に | v1 もここが核心、アプローチが異なる |
| 4 | トップページをマストヘッド方式に。フィーチャーコピー表示 | v1 はスライダー化 |
| 5 | 「DATA BOX」サイドバー整備。成績をインラインで言及するコピー生成 | v1 にない新機能 |

## v1 vs v2 比較表

| 観点 | v1「深夜の円卓、かすかな光」 | v2「黒革の会員誌」|
|------|---------------------------|-----------------|
| 参照元 | VTuber事務所・ゲームサイト（ぶいすぽっ！・メメントモリ） | 高級クラブ誌・競馬新聞・エディトリアルデザイン |
| 主役アクセント | キャラカラー（グロー） | ゴールド（ライン・ルール） |
| 背景色 | `#0e0e12`（チャコール） | `#0c0c0d`（漆黒） |
| 英字フォント | Playfair Display | EB Garamond |
| 日本語フォント | 源ノ明朝 / Noto Serif JP | Shippori Mincho B1 / Noto Serif JP |
| 本文フォント | Noto Sans JP（サンセリフ） | Noto Serif JP（セリフ統一） |
| ホバー演出 | グロー + 浮き上がり | ゴールドボーダー強化のみ |
| キャラヒーロー | フルブリード立ち絵スライダー | 雑誌見開き（テキスト左・画像右） |
| セリフ表現 | blockquote → 大型化（1.5〜2rem） | pull-quote（縦ライン付き、2〜2.5rem） |
| グロー効果 | 多用（カード・ヒーロー・ボーダー） | なし（格式感を優先） |
| 印象 | 没入型・インタラクティブ・エネルギッシュ | 重厚・読み物・格式・静的 |
| 向いているユーザー | VTuber・ゲームに親しんでいる層 | 競馬ファン・読み物が好きな層 |
| 実装コスト | 中〜高（スライダー・アニメーション） | 低〜中（主に CSS + タイポグラフィ） |

## どちらの案を選ぶか

**v1 を選ぶべき場面**: キャラクターに深く感情移入してほしいとき。VTuber的な関わりを促したいとき。  
**v2 を選ぶべき場面**: 競馬の格式・重厚感を前に出したいとき。「読む」体験を大切にしたいとき。

**ハイブリッド案**: v1 の「キャラヒーローのフルブリード」をベースにしつつ、v2 の「ゴールドルール・大文字ナビ・pull-quote」を重ねることも可能。  
フェーズ 1（フォント・カラートークン）はどちらを選んでも同じ基礎になるため、まずそこから着手すると判断が後からできる。
