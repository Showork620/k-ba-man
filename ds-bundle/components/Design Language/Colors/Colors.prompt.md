# Colors — k-ba-man カラートークン

CSS カスタムプロパティで管理するダーク×ゴールドのカラーシステム。

## 背景 / サーフェス
| トークン | 値 | 用途 |
|---|---|---|
| `--bg` | #0b0c11 | ページ背景 |
| `--surface` | #111319 | カード・パネル |
| `--surface-2` | #181a24 | ホバー・ネスト |
| `--surface-3` | #20222e | 深いネスト |

## テキスト
| トークン | 値 | 用途 |
|---|---|---|
| `--ink` | #edeae0 | 見出し・強調 |
| `--text` | #a8a59c | 本文 |
| `--muted` | #62647a | ラベル・メタ |
| `--faint` | #3a3b4e | 非活性 |

## ゴールドアクセント（主要ブランドカラー）
| トークン | 用途 |
|---|---|
| `--gold` | ボタン背景・ランク番号・ハイライト |
| `--gold-light` | ホバー時 |
| `--gold-dim` | 選択状態背景（不透明10%） |
| `--gold-glow` | グロー効果（不透明18%） |

## ボーダー
- `--border` #1c1e2c — カード外枠
- `--border-mid` #252738 — インタラクティブ要素

## セマンティック
- `--green` #26845a — 利益・成功
- `--amber` #b87400 — 警告・注意
- `--red` #be3636 — 損失・エラー

## キャラクターアクセント `--accent`
各キャラクター固有のカラー。コンポーネントに `style="--accent: #7f1d1d"` でインライン指定します。
龍之介 `#7f1d1d` / 誠 `#0f766e` / 美咲 `#6d28d9` / 健太 `#15803d` / 鉄平 `#a16207` / さくら `#be123c` / 葵 `#1d4ed8` / 陽菜 `#ea580c` / 優子 `#047857` / 吾郎 `#854d0e`
