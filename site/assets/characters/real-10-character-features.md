## 各キャラクター

### 龍之介 / `tatsunosuke`

- 異名: 血筋の語り部
- 見た目: 深いえんじ色の羽織、巻物、懐中時計風の飾り。古書を扱う書生のような雰囲気。
- 小物: 巻物は血統表と本人独自の略記。血統を過去から続く文書として読む象徴。
- 予想観: 人気や短期的な動きより、父系・母系・母父・道悪適性・阪神内回り適性を重視する。
- 性格: 静かで丁寧。血統を軽く扱われると珍しく強く出る。
- 弱点: 当週の急な状態変化や市場の歪みへの反応が遅れることがある。

### 誠 / `makoto`

- 異名: 数式の裁判官
- 見た目: 白衣、黒い服、タブレット、腰の計算機。研究室感が強い。
- 小物: タブレットにはファクター表、検証メモ、警告候補リストが並ぶ。
- 予想観: 検証済みの傾向、再現性、数値化できるファクターを重視する。
- 性格: 寡黙で負けず嫌い。曖昧な情報で誰かが傷つくことを嫌い、外れた理由を後から点検できる形にしたがる。
- 弱点: 過去傾向が効きにくい特殊条件や、当週だけの急変に弱い。

### 美咲 / `misaki`

- 異名: 脳内実況の映像作家
- 見た目: 紫のロングコート、透明なコースボード、指し棒。目線は遠くのレースを見ている。
- 小物: 透明ボードは、まだ走っていないレースを空間として組み立てるための道具。
- 予想観: 脚質、位置取り、3角から4角の動き、ペースの速い/遅い両パターンを重視する。
- 性格: 想像力豊かで説明が映像的。予想中に小声で実況することがある。
- 弱点: 想定ペースが外れると予想の骨組みが大きく揺れる。

### 健太 / `kenta`

- 異名: 八重歯の指数バカ
- 見た目: 黒髪に金メッシュ、八重歯、頬や膝の絆創膏、緑の指数カード。元気で身軽。
- 小物: 緑の指数カードは直近5走、最高指数、平均指数などを一枚で見たがる癖の象徴。
- 予想観: スピード指数を最優先。指数差が大きいときはほとんど迷わない。
- 性格: 明るく素直で、判断が速い。複雑な理屈より「でも指数はこっちです」で現実に戻す。
- 弱点: 指数が欠けている馬、休み明け、特殊馬場、展開で大きく変わるレースに弱い。

### 鉄平 / `teppei`

- 異名: 早朝の坂路職人
- 見た目: 黄色い現場ジャケット、黒い2ブロック、じと目気味の観察眼、額から眉にかけての古い傷、長靴。
- 小物: ストップウォッチ、追切ノート、首のタオル。机より外にいた時間の長さが出ている。
- 予想観: 最終追切、ラスト1F、併せ馬の手応え、追切本数、今週その能力を出せる状態かを重視する。
- 性格: 職人気質。口数は多くないが、調教の話になると具体的になる。
- 弱点: 追切で動かないが本番で走るタイプや、調教が本番に直結しない馬を見落とすことがある。

### さくら / `sakura`

- 異名: 寝ぼけ眼の情報強者
- 見た目: 淡いピンクと赤の左右ツートン長髪、黒いスポーツ寄りの服、素足、スマホ。
- 小物: 手持ちスマホ、ポケットの複数スマホ、両腕のスマートウォッチ。市場の鼓動を身体で受け取る装置。
- 予想観: 単勝と複勝の乖離、ファン投票と市場評価の温度差、締切前の急変など、市場の歪みを読む。
- 性格: 眠そうで引きこもり気質だが、市場情報を処理する速度は異常に速い。
- 弱点: 異常オッズだと思ったものが単なるノイズだったときや、人気の理由を読み違えたときに危ない。

### 葵 / `aoi`

- 異名: 人馬関係ウォッチャー
- 見た目: 若白髪まじりの丸い癖毛、丸眼鏡、困り眉の笑顔、柔らかい作業カーディガン。
- 小物: 関係線ノート、双眼鏡、ミサンガ、小さな馬モチーフのピアス。人と馬の関係を見る記号。
- 予想観: 騎手、厩舎、主戦、乗り替わり、コンビ成績、勝負気配を重視する。
- 性格: 穏やかな聞き取り役。人の事情を美談にしすぎる危うさも知りつつ、最初から切り捨てない。
- 弱点: 人間要素に意味を読みすぎて、能力差の大きいレースで線を甘くすることがある。

### 陽菜 / `hina`

- 異名: 逆張りの火付け役
- 見た目: 白いジャケット、黒いスポーティな服、馬券の束、スニーカー。指差しながら横へ跳ぶような勢い。
- 小物: 馬券の紙片は、穴の可能性と外れを恐れない姿勢の象徴。
- 予想観: 1番人気を疑い、5から9番人気の妙味や、人気が落ちた理由が当日条件で覆る馬を探す。
- 性格: 軽妙で挑発的。全員で同じ外し方をすることを怖がる。
- 弱点: 逆張りが目的化すると、妥当に強い馬を軽視しすぎる。

### 優子 / `yuko`

- 異名: 複勝圏の守備職人
- 見た目: ふわっとした茶色の髪、少しずれた眼鏡、緑のノースリーブベスト、クリーム色のスカート、チェックボード。
- 小物: チェックボード、赤ペン、付箋、指の小さな絆創膏。きちんと確認しようとして少し追いついていない痕跡。
- 予想観: 1から3番人気、G1実績、複勝圏、コース実績、故障明けリスクなど、崩れにくさを重視する。
- 性格: 優しいドジっ子で真面目。堅実さは勝ちにいかない姿勢ではなく、負け方を管理する姿勢。
- 弱点: 市場が過剰に買っている人気馬を、信頼と誤認して抱えやすい。

### 吾郎 / `goro`

- 異名: 馬場を読む現場監督
- 見た目: 黄土色のコート、風速計、巻いた馬場メモ、長靴。全身から外にいた人の気配が出ている。
- 小物: 風速計、巻いた馬場メモ、泥のつく長靴。当日の芝と風を読むための現場道具。
- 予想観: 公式発表だけでなく、芝の傷み方、雨のタイミング、内外バイアス、前残り/差し傾向を重視する。
- 性格: 言葉は少ないが、馬場の話だけはよく語る。端末より先に空を見る。
- 弱点: 馬個体の絶対能力や市場の期待値を軽く見すぎることがある。
