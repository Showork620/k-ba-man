# 買い目ランの記録規約

## r2（第2ラン）は公式記録から除外

`bets/r2/` および `bets/aggregated-r2.json` は **ダブルチェック専用**（同通確認・方向性検証など）。
実際の損益計算・採点・累積記録には **使用しない**。

- 公式記録の対象: `bets/v1/` + `bets/aggregated-v1.json` のみ
- `score-race.mjs` の損益集計・採点カードは v1 ベースで行う
- r2 の結果を会話で言及する場合も「参考情報」として明示し、記録には混入させない

## ラン命名の意図

| ディレクトリ | 用途 |
|---|---|
| `bets/v1/` | 公式・正式な買い目（記録対象） |
| `bets/r2/` | 同通確認・サニティチェック専用（記録対象外） |
